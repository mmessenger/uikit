Application Bar

The application bar provides a location for placement of corporate identity information, product name, the user menu, optional global functions, and navigation to major modules of the application.

Overview
An application bar should be used on every standard page

Flyout Navigation

Flyouts Navigation slide down from the header and slide over the page contents. They can be closed by clicking on the selected Button within the header or anywhere outside the flyout. Upon closing, they slide up into the header. When the flyout navigation is used on smaller screen the flyouts are presented in line with other navigation links and can be expanded/collapsed.

Mega Menu Navigation

            
Mega Menu Navigation are often used on large-scale sites that have many potential pages for users to navigate to. The mega menu opens up an expanded area that offers more options.

Tabbed Navigation

            
Tabbed navigation consists of buttons or tabs aligned side by side and it is usually located at the top of the page. Tabbed navigation is a good choice when the number of tabs is not likely to change often and the tabs fit on one row. The navigation element should span across the entire width of the page using limited as well as short and predictable titles with the current selected tab clearly highlighted.

Min-cart View

            
When users click on shopping cart icon, it provides a mini-cart view of the shopping cart. This view displays 3 recently added items. This view eliminates the steps in the shopping process where users can confirm what itmes are just added in cart and make any selections or deletions. The View All link or Check Out button direct users to checkout process.
