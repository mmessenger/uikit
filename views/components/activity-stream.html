Activity Stream

The activity stream displays upcoming, current, and past activities.

Overview

The Activity Stream is an event-based timeline meant to keep the user informed of activity within the application. Activity streams promote user engagement and expose messages, events and transactions that have recently been triggered by the system, user(s), or external applications.
