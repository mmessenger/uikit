Page Header

Page Headers, displayed at the top of several pages, are used to provide a high-level overview for the page. Generaly composed of multiple components, page headers can include a heading, details, snapshots, and a minimal set of primary actions or controls related to the content.

Overview

Text

When to Use:
  • On each main-level module screen of the application to provide positional cues and a sense of place.
  • When there's a relevant insight to help enhace a users understanding of the content.
  • When the user needs to act on the entire page content.

Dashboard Header

The page header is located at the top of a dashboard screen. It includes the dashboard title, a short description, the last updated time and date, refresh and a method for editing the content displayed on the dashboard.

Module Header

            
The page header is located at the top of every main application module (e.g. customers, inventory, sales). It includes the module title, a short description, the ability to toggle views as well as the context of the module, and other related actions.

Record Header

            
The page header is displayed when the user selects a single record and is located at the top of record home. It includes the record title, navigation back to the module list and other related actions. In addition the record header contains a horizonal row containing details or snapshots related to the record. Record headers may be collapsed or expanded accompanied by a slide transition.

Related List Header

            
The Related List Header is display when a user selects a list related to a single record (e.g. view related records). It includes a list title, a breadcrumb navigation back to the orignal module and record, a short description of the list and other related actions.

Dialog Header

            
The dialog header is displayed at the top of a modal dialog when a new object is being created. A short description is displayed for users who need further explanation as to the screen requirments (why is the information be requested, what can I do here). The short descrition may be expanded or collapsed.

Search Results Header

            
The serach results header is displayed at the top of search results page. It includes the page title, a seach keyword, and the number of records found.
