Sidebar

Sidebar is a vertical area that displays onscreen and presents widget components and navigation in a text-based hierarchical form. On web applications, the sidebar navigation should be open by default, and navigation items should only link to screens within the application (no external links). If secondary items exist within a navigation item, the top-level item is not an actual destination. Clicking a navigation item with secondary navigation expands that itemâ€™s secondary navigation. The preferred format for navigation items is Title Case

When to Use:
  • To access all main high-level sections of application.
  • To access to Home/top-level of the application.
  • To provide sense of place to the user (display the depth and breadth of the app/site.)
Sidebar Classes

The sidebar is css-driven - and various classes can be combined to control it's width and color. Below you'll find summarized tables with the available sidebar navigation classes.

Class Description
.nav-sidebar Default sidebar class, added to nav container.
.nav-sidebar.bg-* Defines sidebar background color. According to the custom color system, sidebar background color can be changed to one of the available colors by adding a proper class to the main sidebar container.
.nav-sidebar.nav-sm Small sidebar has 100px width, with stacked icons and text. Second navigation levels and sublevels opens on parent level hover. It is required to add .nav-sm class to the along with the .nav-sidebar class.
.nav-sidebar.nav-xs Mini sidebar has 70px width, with icons only. Second navigation levels and sublevels opens on parent level hover. It is required to add .nav-xs class to the along with the .nav-sidebar class.
.hidden-nav Hides content when the sidebar is small or mini
.visible-nav Displays content when the sidebar is small or mini
data-toggle="class:nav-sm" Collapsible sidebar. Changes main sidebar width from default to small. It is required to add a control along with a data-target="#sidebarID".
data-toggle="class:nav-xs" Collapsible sidebar. Changes main sidebar width from default to mini. It is required to add a control along with a data-target="#sidebarID".
