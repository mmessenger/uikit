Wizard

Wizards break down tasks or information into a sequence of chucks, navigated with Previous and Next buttons and using sequence labeling to indicate where a user is in the process and how much work is required.

When to Use:
  • To present a fixed workflow in a prescribed sequence.
  • To break up a long or complex workflow into manageable tasks.
  • To disclose information upon what data a user has already provided.
Basic Wizard

Wrap .steps-container class around a list of .steps and .step-content

Classess
Class Description
.steps-container Initializes the wizard containing .steps, .steps-content and navigation buttons.
.steps Indicates placement for step indicators and adds default styling.
.step-content Wrap all of the step panes
.step-pane Use this class for each pane and data-step to indicate the sequence of the pane. For instance, data-step="1"
.btn-next Indicates the next button
.btn-prev Indicates the previous button
