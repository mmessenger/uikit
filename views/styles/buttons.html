Button Colors

The base .btn class defines the button element and looks like a plain text link. Adding a .btn-* class will define the button styling. In addition to the standard bootstrap symantic colors you may add a .btn-default, .btn-white, .btn-dark, and .btn-black.



            
Button Outlines

            
Use the .btn-outline-* class instead of .btn-* for a outline button. You can increse the border size using .b-2x, .b-3x, .b-4x or .b-5x. In addition to the standard bootstrap symantic colors you may add a .btn-default, .btn-white, .btn-dark, and .btn-black.

Button Widths

            
Buttons are as wide as their text. For specific sizing use .btn-w-xl, .btn-w-lg, .btn-w-md, .btn-w-sm or .btn-w-xs 

            

            


            
Responsive Widths

            
Add a .btn-block to acheive a reponsive button. You may also leverage various sizes.
