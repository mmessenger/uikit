Avatars

Avatars are used to provide a visual representation of a person or a group representing persons. When a personalized avatar isn’t available for a single object, the objects initials or placholder image will be displayed in it’s place.

Overview

An avatar can be square, circular or a rounded rectangle, depending on usage. The default is a cirlce. Use a circle for all people-oriented objects that could potentially render as avatars.

Classes

            

                
                    

                        Class
                        Description
                    

                
                
                    

                        .avatar
                        Defines the avatar
                    

                    

                        .avatar-text
                        Applies styles to present user initials when no image 
                    

                    

                        .avatar-*
                        Defines the size of the avatar. The default is 50x50 pixels. Availble in the following sizes: -lg, -md,-sm,-xs 
                    

                    

                        .bg-*
                        Defines avatar background color. According to the custom color system, avatar background color can be changed to one of the available colors by adding a proper class.
                    

                    

                        .img-*
                        Defines avatar size. The default is square. Availble in the following formats: -rounded, -circle,-thumbnail
                    

                    

                        i
                        Defines the avatar indicator. The indicator is automatically positioned top and left
                    

                    

                        .bottom
                        Positions the indicator at the bottom
                    

                    

                        .top
                        Positions the indicator at the top
                    

                    

                        .right
                        Positions the indicator at the right
                    

                    

                        .left
                        Positions the indicator at the left
                    

                    

                        .md
                        Increases the size of the indicator
                    

                    

                        .on
                        Colors indicator with .bg-success
                    

                    

                        .off
                        Colors indicator with .text-muted
                    

                    

                        .away
                        Colors indicator with .bg-warning
                    

                    

                        .busy
                        Colors indicator with .bg-danger
