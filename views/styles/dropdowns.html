Dropdowns

A dropdown allows a user to select a value from a series of options.

Basic Dropdown

A Drop Down is used when the user needs to select from a few item or no more than 25 items. For larger selection list use a drop down with Type Ahead. Dropdown capitalization is sentence case, except when the list displays formal names.

Dropdown With Icons

Icons can be included on either the right, left, or both sides of an option.

Dropdown Positioning

A dropdown may also be positioned top or bottom.

Dropdown Submenus

A dropdown may have one or more submenus levels and .dropup or .pull-left

Dropdown Classes
Class Description
.dropdown Initializes dropdown. The target HTML element and dropdown need to be wrapped in this class
.dropdown-toggle Enables dropdown to show on click
.dropdown-menu Identifies the item menu dropped
.dropdown-header Styles a header to label sections of actions
.dropdown-item Initializes dropdown item. Can be a <a> or <button>
.dropdown-divider Separate groups of related menu items with a divider.
.dropdown-submenu Allows for multilevel dropdowns and designates the item as a submenu
.dropup Changes the dropdown to expand upwards when added to the parent container.
.active Applies active state to dropdown item
.disabled Styles dropdown items as disabled.
.arrow Adds and arrow to the designated position on the dropdown.
.arrow.left
.arrow.bottom		 Positions the arrow above or below the target. The default location is center, but can be combined with .pull-right and .pull-left for left/right alignment
