Lists

A list is used to group related records

Overview

Supports two basic List types: Basic lists and list groups. Lists contain a set of items displayed in rows, where each item is represented as a text label. Depending on the purpose of the list, each row may also be accompanied by a control. The user can also select a row to perform an action on the corresponding item.

When to Use:

  • When you need to present a group of related records, space is limited and the user does not need robust sort/filtering
  • hen you need to present a group of related records, and the user does not need to compare items in sequence.

Classes

            

                
                    

                        Class
                        Description
                    

                
                
                    

                        .list-inline
                        Defines a basic list as inline
                    

                    

                        .list-inline-item
                        Changes list item to inline element
                    

                    

                        .list-unstyled
                        Removes basic list formating
                    

                    

                        .list-group
                        Defines the list group. Adds basic default styling.
                    

                    

                        .list-group-lg
                        Increases the padding and font sizing of the list items
                    

                    

                        .list-group-sm
                        Decreases the padding and font sizing of the list items
                    

                    

                        .list-group-sp
                        Separates each list item with a margin
                    

                    

                        .list-group-alt
                        Alternates backgrouds styles for repeating list items. May be used with backgroud colors.
                    

                    

                        .list-group-flush
                        Removes the default radius and side borders from the list group. Helpful when embedding lists in a bordered container such as cards.
                    

                    

                        .bg-*
                        Use modifier classes to change the appearance of each step by adding .bg-* to the .steps class. You may also apply variations by adding .dk, .dker, .lt, .lter to darken or lighten the base color.
                    

                    

                        .no-borders
                        Removes all borders from the list group
                    

                    

                        .no-border
                        Remove the list group outside border
                    

                    

                        .list-group-item
                        Defines the list group. Adds basic default styling.
                    

                    

                        .list-group-item-*
                        According to the theme color applies contextual color to the list item.
                    

                    

                        .list-group-item-heading
                        Defines default styling for the heading
                    

                    

                        .list-group-item-text
                        Defines the base text of the list item
                    

                    

                        .active
                        Applies active styling based on the color
                    

                    

                        .disabled
                        Applies disabled styling to the item based on the color
