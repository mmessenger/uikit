Tabs

Tabs are used to separate content into different views, which can be displayed one at a time without a page refresh. Tabs provide easy access to additional content without requiring the user to go to a new page.

Overview
  • Use tabs when content is closely related, but mutually exclusive
  • Use tabs when it is not important to see more than one content pane at a time
  • Always show one tab as selected with its content visible.
  • Selecting a tab will not affect other parts of the page.
  • Tabs should always appear in the same location as the user clicks from one tab to another.
  • Limit labels to short words â€“ try not to wrap
  • Capitalization of horizontal tab names is Title Case

Classes

            

                
                    

                        Class
                        Description
                    

                
                
                    

                        .nav-tabs
                        Initializes default tabset. Also requires base .nav class.
                    

                    

                        .nav-inline
                        Spaces nav links horizontally.
                    

                    

                        .nav-justified
                        Makes tabs equal widths of their parent container on larger screens. On smaller screens the nav links are stacked.
                    

                    

                        .nav-center, .nav-right
                        Positions the tabs as indicated. The default tab position is left.
                    

                    

                        .nav-item
                        Styles each list item as a single tab
                    

                    

                        .nav-link
                        Styles each link element in the list item
                    

                    

                        .dropdown
                        Used on a tab containing an overflow dropdown menu.
                    

                    

                        .tab-content
                        Styles each tab content wrapper
                    

                    

                        .tab-content-bordered
                        To acheive a scoped tab apply border and padding to the tab content.
                    

                    

                        .bg-*
                        Defines tab background color. According to the custom color system, sidebar background color can be changed to one of the available colors by adding a proper class to the main tab container
                    

                    

                        .active
                        Applies the active state to a tab list item. A tab is hidden by default, and will only become visible when given the class name active.
                    

                    

                        .disabled
                        Disable any tab item by adding .disabled class to the nav-link
                    

                    

                        .show
                        Shows the tab
                    

                    

                        .hide
                        Hides the tab
