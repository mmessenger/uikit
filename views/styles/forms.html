Form Elements

Forms allow a user to enter and read data.

Overview

Though we try not to, sometime we need to ask users to enter information. Form elements and controls help us capture this information. Forms are the primary place where users manage their data and complete tasks in an application. For example, forms can be used to enter shipping or credit card data to order a product. It is very important that applications use consistent controls to manage users expectations about how an application will work.

Input
Checkboxes & Radios

Checkboxes are for selecting one or several options in a list, while radios are for selecting one from many.
