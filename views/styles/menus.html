Menus

A menu displays grouped navigation actions

Overview

The menu is a flexible, all-purpose component for navigation. The menu bar is a wrapper of the menu and can be oriented horizontally or vertically and positioned off-screen.

When to use:
  • To access all main high-level sections of application.
  • To access to Home/top-level of the application.
  • To provide sense of place to the user (display the depth and breadth of the app/site.)

Classes

Class Description
.nav Initializes default nav. Responsible for basic navigation styling.
.nav-item Styles each list item as a menu item.
.nav-link Styles each link element in the menu item
.nav-inline Space out nav links in a horizontal band
.nav-inline Space out nav links or items in a horizontal band
.nav-pills Styles each nav link or item in rounded blocks. Provides active and hover states
.nav-stacked Vertically orientation for the default nav
.navbar Initializes default navbar.
.navbar-header Navbar can be placed in a header. Adds active states and makes menu item flush.
.header-md .navbar Increases the height of the horizontal navbar to 60px
.navbar.bg-* Defines navbar background color. According to the custom color system, navbar background color can be changed to one of the available colors by adding a proper class to the main navbar container.
.navbar-sidebar Navbar can be placed in a sidebar.
.nav-off-screen Makes the nav visible as an off-screen sidebar.
.disable Disable any navbar navigation item.
.active Applies active state to menu item.
.dropdown Place a nav inside a default dropdown
